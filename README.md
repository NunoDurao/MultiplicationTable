# MultiplicationTable 

## Description
MultiplicationTable is a simple program developed using HTML, CSS, and JavaScript that generates a multiplication table.

## Features
- Allows users to select a number and generate the multiplication table for that number.
- Clean and intuitive user interface.
## How to Use
1. Clone or download this repository to your local machine.
2. Open the `index.html` file in your web browser.
3. Enter a number in the input field and press the "Generate Table" button.
4. The multiplication table for the entered number will be displayed below the input field.

## Technologies Used
- HTML
- CSS
- JavaScript
